SEC 120B Introduction to Penetration Testing with Burp

This course is an introduction to web penetration testing using the Burp Suite offered by PortSwigger. It covers how to test for the common web application vulnerabilities introduced in the SEC 101 course.

Title Introduction to Penetration Testing with Burp
Code SEC 120B
Topics Download, install, run Burp
Configuration for intercepting HTTPS traffic
Intercept and modify browser traffic
Replay web requests
API
Automation
Requirements Participants will need to bring their own computers capable of running Burp (a Java application) and a modern web browser.
Duration 8 hours
Audience Quality assurance engineers
Software developers
